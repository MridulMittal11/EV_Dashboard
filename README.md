# Electric Vehicle (EV) Dashboard

## 📌 Overview
This project presents an interactive dashboard focused on analyzing electric vehicle (EV) trends, adoption, and performance. It provides insights into EV growth patterns, vehicle distribution, and regional adoption using visual analytics.

## 🛠 Tools & Technologies
- Tableau / Power BI  
- Data Visualization  

## 🔍 Features
- Interactive filters and slicers for dynamic analysis  
- Visualization of EV adoption trends over time  
- Comparison of vehicle types and categories  
- Geographic distribution of electric vehicles  
- Clean and user-friendly dashboard design  

## 📊 Key Insights
- Significant growth in electric vehicle adoption over recent years  
- Variation in EV usage across different regions  
- Certain vehicle types dominate the market  
- Clear upward trend in EV popularity and demand  

## 🚀 Conclusion
This project demonstrates the ability to build interactive dashboards and communicate insights effectively by transforming raw data into meaningful visual stories. It highlights strong skills in data visualization and analytical thinking.
